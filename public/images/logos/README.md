# Logos de TAX DUITAMA S.A.S.

Coloca aquí las imágenes de los logos de la empresa.

## Archivos requeridos:

1. **logo-completo.jpeg**
   - Logo completo con texto "TAX DUITAMA S.A.S.", corona, escudo, alas y banner
   - Se usa en el Footer
   - Tamaño recomendado: mínimo 600px de ancho

2. **logo-simple.jpeg**
   - Logo simplificado para el Header
   - Puede ser solo el escudo o escudo con texto
   - Tamaño recomendado: altura máxima 60px

## Notas:
- Las imágenes se cargarán automáticamente desde esta carpeta
- Si las imágenes no existen, el componente mostrará un fallback
- Asegúrate de que los nombres de archivo coincidan exactamente:
  - `logo-completo.jpeg`
  - `logo-simple.jpeg`
